# Privacy Policy

Last updated: 2026-04-08

Volume Booster 300 does not collect, transmit, sell, or share personal data.

## Data stored locally

- The extension stores per-page volume settings in Chrome's local extension storage.
- The extension stores a local default volume setting.
- The extension may store a local theme preference.

## Data use

- Stored values stay on the user's device.
- No account is required.
- No analytics, tracking, advertising, or remote code is used.

## Network access

- The extension does not send user data to external servers.
- The extension does not use remote configuration or cloud sync.

## Permissions

- `storage` is used to save local settings.
- `tabs` is used to read the active tab so the popup can target the current page.
- `windows` is used to open the optional popout window.

## Contact

For questions about this extension, contact the developer through the Chrome Web Store listing or the support contact provided there.
