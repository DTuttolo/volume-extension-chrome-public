# Chrome Web Store Listing

## Name

Volume Booster 300

## Short description

Boost tab audio up to 300% with per-page memory and a compact popout window.

## Full description

Volume Booster 300 is a lightweight browser audio booster for pages with media playback.

Features:

- Boost audio from 0% to 300%
- Save volume per page
- Quick presets for MUTE, 100%, 150%, 200%, 250%, and 300%
- Optional popout window for a portable standalone view
- Dark and light theme support

Notes:

- This extension only affects audio inside Chrome.
- Values above 100% may cause clipping or distortion.
- Settings are stored locally on your device.

Privacy policy link:
