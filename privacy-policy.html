Privacy Policy

Last updated: 2026-04-09

Volume Booster 300 does not collect, transmit, sell, or share personal data.

Data Stored Locally

  • The extension stores per-page volume settings in Chrome local extension storage.
  • The extension stores a local default volume setting.
  • The extension may store a local theme preference.

Data Use

  • Stored values remain on the user device.
  • No account is required.
  • No analytics, tracking, advertising, or remote code is used.

Network Access

  • The extension does not send user data to external servers.
  • The extension does not use remote configuration or cloud sync.

Permissions

  • storage: saves local settings.
  • tabs: reads the active tab so the popup can target the current page.
  • windows: opens the optional popout window.

Contact

For privacy questions, contact the developer through the Chrome Web Store support contact.
